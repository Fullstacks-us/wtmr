About This Application

A lightweight, web-based system for logging water tests, meter readings, and mileage records.

Features:

  • Water test logging (pH, Chlorine, Hardness)
  • Water meter readings with delta calculation
  • Mileage tracking with billable flags
  • History view with filtering and export

Note: This is a demo deployment. For full functionality, deploy the backend server separately.
